# Sprint 2 - PIM Module Test Scenarios

## Module: PIM

### Scenario 1: Navigate to PIM Module
Verify that admin user can navigate to the PIM module from the sidebar.

### Scenario 2: Open Add Employee Page
Verify that admin user can open the Add Employee page from PIM.

### Scenario 3: Add New Employee
Verify that admin user can add a new employee with first name, last name, and unique employee ID.

### Scenario 4: Search Employee by Employee ID
Verify that admin user can search the newly created employee using Employee ID.

### Scenario 5: Open Employee Personal Details
Verify that admin user can open the searched employee details page.

### Scenario 6: Edit Employee Gender
Verify that admin user can update employee gender and save the details.

### Scenario 7: Delete Employee
Verify that admin user can delete the created employee from Employee List.
