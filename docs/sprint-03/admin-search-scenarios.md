# Admin Module - Search User Test Scenarios

## Feature
Search System User

## Module
Admin

## Preconditions
- User is logged in as Admin.
- User is on the Admin > System Users page.

---

## Test Scenarios

### TS-ADMIN-SEARCH-001
Verify Admin can search an existing system user by Username.

### TS-ADMIN-SEARCH-002
Verify search returns matching user details.

### TS-ADMIN-SEARCH-003
Verify search with non-existing username displays "No Records Found".

### TS-ADMIN-SEARCH-004
Verify Reset button clears all applied search filters.

### TS-ADMIN-SEARCH-005
Verify search works after multiple consecutive searches.

### TS-ADMIN-SEARCH-006
Verify Username search field accepts valid input.

### TS-ADMIN-SEARCH-007
Verify search functionality is accessible after page refresh.
