# Login Test Scenarios

## Functional

- Verify login with valid credentials
- Verify login with invalid username
- Verify login with invalid password
- Verify login with invalid username and password
- Verify required validation for empty username
- Verify required validation for empty password
- Verify required validation when both fields are empty
- Verify successful logout
- Verify session persistence using Storage State
- Verify unauthenticated users are redirected to Login

## Automation Scope

- ✅ Valid Login
- ✅ Invalid Login
- ✅ Required Field Validation
- ✅ Logout
- ✅ Storage State
